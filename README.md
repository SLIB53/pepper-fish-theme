# Pepper Fish Theme

_Pepper_ is a fish theme showing an indicator for the system, the current working directory, git status, and a bullet for the prompt indicator.

![screenshot-ghostty-peppermint](screenshot.png)


## Requirements

* git


## Installation & Usage

Run the [“apply theme” script](./scripts/apply_theme.sh) from fish:

```sh
fish scripts/apply_theme.fish
```

**WARNING:** This will clobber your existing prompt.
